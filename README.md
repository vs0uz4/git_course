# Curso de Git

Exercício do curso de git.
Descrição de todas as instruções usadas para fazer o push do repositório local, desde o commit.

## Instruções Utilizadas No Push do Repositório

- Commitando o arquivo recem adicionado 'README.md';

```git commit -m 'Adicionando arquivo Readme.md'```

- Adicionando o repositório remoto;

```git remote add origin https://github.com/vs0uz4/git_course.git```

- Subindo o repositório para o GitHub com `push`;

```git push origin master```
